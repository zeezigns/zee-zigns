🔔 Order Notifications

Choose how you want to be notified when a new order is placed:

📱

WhatsApp via CallMeBot RECOMMENDED

Free & instant. Set up a WhatsApp API key at callmebot.com, then add your phone number and API key in the server's .env file as WA_PHONE and WA_KEY. The server will ping your WhatsApp on every new order.

✉️

Email (Nodemailer + Gmail)

Send yourself an email alert on each order via Gmail. Add GMAIL_USER and GMAIL_PASS (App Password) to your .env file — the server will auto-email you.

🔔

Browser Push (ntfy.sh)

Subscribe to a private channel at ntfy.sh — completely free. Set NTFY_TOPIC in your .env file and install the ntfy app on your phone to get instant push notifications.

How to set up .env:
Create a file called .env in your project root with: 

WA_PHONE=923XXXXXXXXX
WA_KEY=your_callmebot_key
# or
GMAIL_USER=your@gmail.com
GMAIL_PASS=your_app_password
# or
NTFY_TOPIC=your_private_channel
